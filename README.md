# VisaGuide.AI 🌍

An AI-powered full-stack web application that helps users retrieve visa-related details for specific countries using Google APIs.

## 🔍 Project Overview

**VisaGuide.AI** is a full-stack application designed to provide users with up-to-date visa information by fetching and parsing data through Google’s APIs and other integrated sources. The app includes both a frontend interface and a backend API service.

## 📦 Tech Stack

### 🔹 Frontend
- HTML5, CSS3, JavaScript
- React.js
- Tailwind CSS (optional)
- Axios (for API calls)

### 🔹 Backend
- Node.js
- Express.js
- Google Custom Search API
- dotenv (for managing API keys)

### 🔹 Tools & Platforms
- Git & GitHub
- VS Code
- Postman (for API testing)
- Google Cloud Console (API access)

---

## 🚀 Features

- 🔍 Search visa requirements by country
- 🌐 Real-time data fetched using Google Custom Search API
- ⚙️ Modular backend for easy integration of additional APIs
- 🎨 User-friendly UI for better experience
- 📡 Error handling and loading states included
